# Calculador de tablas

```
Opciones:
      --help            Muestra ayuda                                 [booleano]
      --version         Muestra número de versión                     [booleano]
      --nt, --numTable  Numero de la tabla                [número] [defecto: 10]
  -l, --limit           Limite de la tabla                [número] [defecto: 10]
      --sh, --show      Mostrar la tabla en consola  [booleano] [defecto: false]
```
